# DLP UI

DLP UI is a frontend for yt-dlp.

## About this folder

This folder contains webui files.

1. Build frontend in the `src/frontend` directory:

```bash
npm run build
```

2. Move the output to this directory:

```bash
mv src/frontend/dist src/backend/webui
```
